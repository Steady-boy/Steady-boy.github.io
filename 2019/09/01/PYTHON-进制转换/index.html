1).hex()会进行进制转换，转换结果会带有0x。而且hex只会处理数字，不能处理字符串。
2).unhexlify()会将十六进制解码，不仅会处理数字，还会处理字符串。
在python2中可以使用hexlify转化字符串转化为十六进制，但是python3中不可以。在python2和3中都可以使用unhexlify，将十六进制转化为字符串。注意使用unhexlify的时候需要将十六进制加双引号。
